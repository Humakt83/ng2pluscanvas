This page is a simple testing field for angular2 + canvas experiments using paperjs.

Canvas vs dom in angular2
    • + Less styles
      + More performance (this is a common opinion not really verified by these tests)
      + Ability to create complex graphics more straightforwardy than with CSS3
      +/- different browsers can cause variety of challenges (as with css styles and dom elements)
      - Some learning curve
      - More code
      - Angular2's component model is more ideal working with dom
      - Interactivity with user is more of a challenge
    • To minify risks
    • Create components in canvas with modularity and reusability in mind - this should make migration to DOM easier when it is necessary
